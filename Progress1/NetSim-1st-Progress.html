📖 Modul Infrastruktur Jaringan AWS

Route 53: Layanan DNS (Domain Name System) global berkemampuan tinggi yang menerjemahkan URL domain menjadi IP target.

CloudFront: Content Delivery Network (CDN) yang mendistribusikan salinan static cache ke lokasi Edge terdekat guna memangkas latensi data.

VPC (Virtual Private Cloud): Wilayah jaringan virtual terisolasi yang mengamankan penempatan server (EC2 instance) menggunakan kontrol Security Group dan Network ACL.
